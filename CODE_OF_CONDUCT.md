# Code of Conduct

`Gemma4SwiftCore` follows the
[Contributor Covenant Code of Conduct, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be kind. Treat every contributor with respect.
- Assume good intent. Ask questions before assuming someone is wrong.
- Critique code, not people.
- English is the default working language. Other languages are welcome
  in casual discussion, but technical reviews and commit messages stay
  in English so everyone can follow along.

If you observe behavior that violates the Contributor Covenant, please
report it via the channels in [SECURITY.md](SECURITY.md). The maintainer
will respond within 7 days.

## Scope

This applies to all project spaces — issues, pull requests, discussions,
and any external communication channels representing the project.

## Enforcement

The maintainer
[Jingyang Ye](https://github.com/yejingyang8963-byte) is responsible for
clarifying and enforcing this Code of Conduct. All complaints will be
reviewed and investigated promptly and fairly. The maintainer will
maintain confidentiality with regard to the reporter of any incident.

For the full text of the rules, the responsibilities, and the
enforcement guidelines, see the canonical
[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
